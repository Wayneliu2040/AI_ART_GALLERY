AI Art Gallery frontend starter

Files:
- login.html: login page
- gallery.html: post-login image gallery page
- styles.css: shared styling
- login.js: demo login logic
- gallery.js: gallery rendering and interactions

How to test locally:
1. Open login.html in a browser.
2. Use demo@aiartgallery.app / Password123!
3. You will be redirected to gallery.html

How to adapt for Azure Static Web Apps:
- Put these files in your frontend app folder.
- Later migrate to React routes/pages if needed.
- Replace demo logic with real fetch calls to your backend REST APIs.
